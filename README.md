# SubmitFlow

Open-source product submission workflow for founders, indie hackers, and SEO builders.

SubmitFlow helps makers submit their products to directories, startup lists, AI tool directories, SaaS directories, and backlink-friendly websites without wasting hours reading submission rules one by one.

It is not a spam backlink bot.  
It is a structured submission CRM and workflow system.

## Why SubmitFlow?

When launching a product, backlinks and directory listings matter.

But the process is painful:

- Some directories are free
- Some are paid
- Some require a badge
- Some offer dofollow links
- Some only provide nofollow links
- Some need manual review
- Some have strict submission rules
- Founders have to fill the same product information again and again

SubmitFlow solves this by helping users manage all product submission tasks in one place.

## Core Features

- Product profile management
- Directory database
- Free and paid submission tracking
- Badge requirement tracking
- Dofollow / nofollow status
- Review time tracking
- Submission status management
- Notes and screenshots
- Backlink status checking
- CSV import / export
- Open adapter system for developers
